# Matchpiece by LUI Playbook

Dashboard interno para administrar pedidos, diseno, produccion, finanzas, delivery, oportunidades de stock y contenido de redes de Matchpiece by LUI Playbook.

## Preview local

Este proyecto esta preparado para Vercel como sitio estatico. Se puede abrir `index.html` directamente o correr un servidor local simple.

## Deploy

1. Subir el repo a GitHub.
2. Importar el repo en Vercel.
3. Framework preset: `Other`.
4. Build command: vacio.
5. Output directory: `.`.

Cuando el dashboard migre a React/Vite con dependencias instalables, el `package.json` ya deja preparada la base para usar `npm run dev` y `npm run build`.
